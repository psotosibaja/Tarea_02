Aspectos importantes

  • Contamos con una capacidad para 500 personas.

  • Para solicitar una cotización puede hacerlo por medio del correo: lastorrejas@outlook.com o xini_1907@hotmail.com
  • El precio del evento varia de lo que la persona desee.
